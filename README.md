# Ford-Johnson Visualizer

An interactive web application that visualizes the **Ford-Johnson (Merge-Insertion) sorting algorithm** step by step.

The project helps understand how the algorithm minimizes comparisons by showing:
- pair formation and ordering
- recursive merging
- Jacobsthal insertion sequence
- comparison flow and data states in real time
